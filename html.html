Processus d'activation:

  1. Ouvrez un compte Sony Network Entertainment sur PSN (ou utilisez votre compte Sony Entertainment Network existant)
  2. Sélectionnez l'icône PlayStation®Store sur l'écran d'accueil de la console PS4™
  3. Une fois sur PlayStation Store, Sélectionnez « Utiliser codes » au bas du menu.
  4. Saisissez le code clé que vous avez reçu dans votre courriel.
  5. Sélectionnez « Continuer » dans la boîte de dialogue.
  6. Sélectionnez « Confirmer » pour accepter les Conditions d’utilisation du code.
  7. Sélectionnez « Continuer » pour terminer l'activation de la carte PSN x USD !
