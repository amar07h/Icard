Comment échanger une carte cadeau Fortnite V-Bucks ?

Une fois que vous avez acheté un code Fortnite V-Bucks, suivez ces étapes pour l'activer :
  1. Cliquez sur Se connecter dans le coin supérieur droit de la page d'accueil de Fortnite
  2. Connectez-vous avec votre compte Epic Games
  3. Passez le curseur de votre souris sur le nom de votre compte dans le coin supérieur droit
  4. Sélectionnez l'option Carte V-Bucks
  5. Appuyez sur le bouton Get Started
  6. Entrez le code clé et cliquez sur Confirmer pour échanger votre carte cadeau Fortnite x V-Bucks !

Good luck in the game! We look forward to your return.
